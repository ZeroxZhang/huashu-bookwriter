# 调研来源指南

书籍创作前必须确认的信息来源清单。

---

## 技术准确性来源

### 官方渠道
| 信息类型 | 来源 | 优先级 |
|---------|------|--------|
| 版本信息 | 官方文档、GitHub Releases | P0 |
| API文档 | 官方API Reference | P0 |
| 安装指南 | 官方Getting Started | P0 |
| 架构说明 | 官方Architecture文档 | P1 |
| 最佳实践 | 官方Best Practices | P1 |

### 社区渠道
| 信息类型 | 来源 | 优先级 |
|---------|------|--------|
| 使用案例 | GitHub Issues、Discussions | P1 |
| 踩坑经验 | Stack Overflow、社区论坛 | P1 |
| 对比评测 | 技术博客、YouTube评测 | P2 |

---

## 市场现状来源

### 竞品分析
| 信息类型 | 来源 |
|---------|------|

| 功能对比 | 官方对比页、第三方评测 |

### 社区热度
| 信息类型 | 来源 |
|---------|------|
| 活跃度 | GitHub Commits、Issues、PRs |
| 社区规模 | Discord/Slack成员数 |
| 内容产出 | 博客文章、教程、视频 |

## 信息验证检查清单

### 版本信息
- [ ] 当前稳定版本号
- [ ] 最近更新时间
- [ ] Breaking Changes历史

### 功能准确性
- [ ] 提到的API是否存在
- [ ] 参数名称是否正确
- [ ] 示例代码是否可运行

### 时效性
- [ ] 信息是否是最新的
- [ ] 是否有过时的建议
- [ ] 是否标注"截至XXXX年X月"
