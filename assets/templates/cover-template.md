# 封面模板

用于PDF封面自动生成。从Markdown文件开头提取元数据。

---

## 标准格式

```markdown
年份月份 · 第X版

## 书名（中文）

副标题 / 英文标题

适用版本：...
模型/技术：...
页数：约XX页 · XX章 + 附录

作者信息
来源说明
```

---

## Type A: 从入门到精通封面

```markdown
2026年4月 · 第2版

## [产品/技术] 从入门到精通

[English Subtitle]

适用版本：[产品] vX.Y.Z
模型：[支持的模型]
页数：约75页 · 10章 + 附录

## 花叔

公众号「花叔」· B站「AI进化论-花生」

本手册基于[产品]官方文档、[来源]编写。
AI工具迭代迅速，部分内容可能随版本更新变化，请以官方文档为准。
```

## 元数据提取规则

| Markdown元素 | 封面位置 |
|-------------|---------|
| 第一个 `# H1` | 封面大标题 |
| H1下方文本行 | 副标题 |
| `**创建者**:` | 作者信息 |
| `**为谁创建**:` | 目标读者 |
| `**基于**:` | 副标题区域 |
| `**最后更新**:` | 日期 |
